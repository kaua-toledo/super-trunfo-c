# Super Trunfo de Cidades (C)

Este é um projeto simples em linguagem C, feito como exercício de programação, usando como tema o jogo Super Trunfo de Cidades.

O programa permite cadastrar duas cartas de cidades, informando alguns dados básicos, e depois exibe essas informações no terminal.

## O que o programa faz

- Cadastra duas cidades
- Lê dados pelo terminal
- Guarda informações como:
  - Estado
  - Código da carta
  - Nome da cidade
  - População
  - Área
  - PIB
  - Pontos turísticos
- Mostra os dados cadastrados de forma organizada

## Como executar

Para compilar e rodar o programa, use:

```bash
gcc super_trunfo.c -o super_trunfo
./super_trunfo
